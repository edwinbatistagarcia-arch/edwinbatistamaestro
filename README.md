# Planificaciones de Lengua Española (5to y 6to de Secundaria)

Sitio web con el currículum docente, las secuencias didácticas, los cuadernos
del maestro y la aplicación de calificaciones para la asignatura de Lengua
Española, 5to y 6to grado del nivel secundario — C.E. Francisco Antonio
Castillo Salcedo.

Este repositorio contiene únicamente los archivos que necesita `index.html`
para funcionar.

## Estructura

- `index.html` — Página principal (currículum, navegación por pestañas y enlaces a todo el material).
- `boletin-app.html` — Aplicación de calificaciones, embebida como iframe en la pestaña "Calificaciones".
- `HTML_Planificaciones/` — Cuadernos del maestro y registros de instrumentos en HTML (creación propia).
- `02_SecuenciasDidcaticas_Originales/` — PDFs de las secuencias didácticas originales.

## Autoría

`index.html`, `boletin-app.html` y todo el contenido de `HTML_Planificaciones/`
son creación propia del docente.

Los PDF en `02_SecuenciasDidcaticas_Originales/` están basados en el formato
y lineamientos curriculares oficiales del MINERD (Ministerio de Educación de
la República Dominicana). Se incluyen aquí únicamente con fines educativos y
de documentación del trabajo docente.

> Nota: la pestaña "Recursos" de `index.html` enlaza a los Libros Abiertos de
> Lengua Española (MINERD) de 5to y 6to, que **no** están incluidos en este
> repositorio por su tamaño (>100 MB cada uno, por encima del límite de
> GitHub). Esos enlaces no funcionarán hasta que se alojen esos PDF aparte
> (por ejemplo, en GitHub Releases o un enlace externo).
